Historical Weather data API

URL format: http://127.0.0.1:5000/api/v1/station/date

URL example of station and date temperature: http://127.0.0.1:5000/api/v1/10/1756-01-02

URL example for one station for all dates: http://127.0.0.1:5000//api/v1/yearly/10/1756

Dictionary data API

URL format: http://127.0.0.1:5000/api/dict/cat
